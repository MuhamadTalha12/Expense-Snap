# Expense Snap 📸✨

Expense Snap is a premium, high-fidelity finance management application designed with a **"Quiet Luxury"** aesthetic. It focuses on minimalist elegance, smooth animations, and effortless transaction entry.

## 🚀 Features

- **Receipt Scanning**: Intelligent extraction of data from receipts using advanced OCR.
- **Voice Entry**: Effortless transaction logging using voice commands.
- **Micro-Animations**: Sophisticated "Bottom Sheet Bloom" and "Gentle Rise with Blur" transitions for a premium feel.
- **Quiet Luxury UI**: A minimalist, high-contrast design system that prioritizes ergonomic interaction and visual clarity.
- **Cross-Platform**: Built with **Flutter**, ensuring a seamless experience on both iOS and Android.

## 🏗️ Project Structure

- `kharcha/`: The main Flutter application source code.
  - `lib/`: Core logic and UI components.
  - `assets/`: Custom fonts, icons, and image assets.
  - `test/`: Comprehensive unit and widget tests.

## 🛠️ Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- Android Studio / Xcode
- A Firebase account (for backend services)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/IamAhmadHabib/Expense-Snap.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Expense-Snap/kharcha
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the application:
   ```bash
   flutter run
   ```

## 🎨 Design Philosophy

Expense Snap follows a "Quiet Luxury" design ethos:
- **Subtle Gradients**: Avoiding harsh shadows in favor of soft, natural gradients.
- **Optical Alignment**: Precise spacing to reduce cognitive load.
- **Haptic Feedback**: Meaningful interactions that feel physically grounded.
